IITC Bookmark to kml

How to use?

  1. Open Ingress Intel Map
  2. Get IITC and Bookmarks for maps and portals plugin and overlay KML plugin.
  3. Add your desired portals to bookmark
  4. In Bookmarks Opt, click Copy bookmarks
  5. Select all and copy the code AND paste under the text area below.
  6. Click Get KML button to download a kml file
  7. Back to Intel Map and display that kml file
